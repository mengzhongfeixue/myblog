# 这是一个简易的博客全栈系统
**该博客系统通过express-generator脚手架创建**
## 使用的技术栈
* node 后端开发环境
* express 后端web框架
* MongoDB noSQL型数据库
* Bootstrap 响应式布局、移动设备优先的HTML、CSS 和 JS 前端开发框架
